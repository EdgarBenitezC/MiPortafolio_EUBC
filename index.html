Acerca de mí

Se llevó a cabo el desarrollo colaborativo de una plataforma web de comercio electrónico dedicada a la venta de productos artesanales mexicanos. En la implementación de esta solución, se emplearon diversas tecnologías de vanguardia, entre las que se destacan CSS y Bootstrap para el diseño y la interfaz gráfica, así como JavaScript para el desarrollo del frontend. Por otro lado, se utilizó Java junto con Spring Boot para el desarrollo del backend.

Para la gestión de la base de datos, se optó por MySQL y Workbench, garantizando un almacenamiento eficiente y seguro de la información.

El proyecto se rigió por la metodología ágil SCRUM, la cual permitió una gestión eficiente y adaptable de los recursos y tareas. Para la coordinación y seguimiento de las actividades, se implementó la plataforma Trello, facilitando la asignación de responsabilidades y el monitoreo continuo del progreso del proyecto.
